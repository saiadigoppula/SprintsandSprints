JavaScript Strings

JavaScript strings are most commonly used data types.

JavaScript strings are used to work with text.

javascript strings can be written in single quotes or double quotes

String Example

Below code will print strings x and y to console

var x = 'GivenSprint';
var y ="Hello World";
console.log(x);
console.log(y);

OUTPUT

//output in console
GivenSprint
Hello World

Quotes inside String

Quotes inside string should not match to outer quotes

var x = 'I "Love" my country';
var y ="This is my first 'Job'";
console.log(x);
console.log(y);

OUTPUT

//output in console
I "Love" my country
This is my first 'Job'

Single , Double quote and Backslash

Single quote should be written as \' .

Double quote should be written as \" .

Backslash quote should be written as \\ .

var single = 'It\'s my job';
var double ="Double quote is written as \" this way";
var backslash ="Backslash is written as \\ this way";
console.log(single);
console.log(double);
console.log(backslash);

OUTPUT

  //output in console
  It's my job
  Double quote is written as " this way
  Backslash is written as \ this way

String concatenation

Combining two strings.

You can concate two or more strings using '+' operator or

String concat() Method.

var x = 'Hello';
var y ='world';
var z = x+y;
var str = x.concat(y);
console.log(z);
console.log(str);

OUTPUT

//output in console
Helloworld
Helloworld

String as Object

String can be object also.

By using new we can create Obeject string.

var x = 'Hello world';
var y = new String("Hello world");

console.log("type of string is :"+typeof x);
console.log("type of string is :"+typeof y);

OUTPUT

//output in console
type of string is :string
type of string is :object
