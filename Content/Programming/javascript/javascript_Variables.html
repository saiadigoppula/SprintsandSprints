JavaScript Variables

JavaScript Variables are used to store the data and values.

var let const are used to decleare Variables

Every variable should be unique

JavaScript Keywords are not allowd to use as variables

JavaScript variables can contain letters, digits, underscores, and dollar signs.

Simple Variables Example

a,b,c are variables declared using var

var a = 5;
var b = 10;
var c = 15;
console.log(a,b,c);

OUTPUT

5 10 15 //output in console

Declaring And Assign

First we decleare a variable then we assign value to it

var name;
name = "kick"
console.log(name);

OUTPUT

kick //output in console
