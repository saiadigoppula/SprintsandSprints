JavaScript Comments

JavaScript comments are used to easily describe the purpose of the code and any algorithms

commented code or text will not executed

you can use comments to prevent execution of block of code

single line and multiline comments are supported javascript

Single-line comments

'//' is used for single line comments

// This line will not executed

var x = 5;           // declared the variable x of value 5

console.log("x"); 
//code prints x to console

OUTPUT

5 //output in console

Multi-line comments

'/*' and '*/' is used for Multi-line comments

code between /* ---- */ will not be executed

/*
 These lines will not executed
 This is Multi-line comment Syntax
 declared the variable x of value 5
 code prints x to console
*/
var x = 5; 
console.log("x");

OUTPUT

5 //output in console

it's a good practice to comment what you are doing
