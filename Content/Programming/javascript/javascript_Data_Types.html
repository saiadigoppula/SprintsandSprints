JavaScript Data Types

Data types refers to types of data that you are going to use .

Data types: numbers, strings, objects ,boolean etc.

Data types tells the type of variable we are using

Data types is an important concept in programming

JavaScript Data Types Example

Below code data types are numbers, strings, boolean 

var a = 25; //number
var b = "kick"; // string
var c = true; // boolean
console.log(a,b,c);

OUTPUT

25 "kick" true //output in console
